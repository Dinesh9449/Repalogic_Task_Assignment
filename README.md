 # Flutter Technical Assignment

This is a Flutter application developed as part of a technical assessment.  
The app demonstrates multiple modules including Weather, To-Do, Game, and Products.

## üì± Features

### 1. Weather Module
- Fetches weather data from an API
- Displays forecast in a ListView
- Allows user to select a city dynamically

### 2. To-Do App
- Add, edit, delete to-do items
- Mark tasks as completed
- Filter tasks by status (All / Completed / Pending)

### 3. Game Module
- Simple Tic Tac Toe game
- Playable between two players
- Game progress is maintained during the session

### 4. Products Module
- Product list loaded from a local JSON file
- Displays product title, price, and image
- Navigate to product detail screen on tap
